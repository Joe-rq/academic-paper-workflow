# 冲突检测与交叉验证

> 本文件是 `paper-review-team` 多 agent 模式**专属**的方法论：在 `.claude/skills/paper-workflow/references/peer-review-simulation.md` Phase 2 浅层"共识 / 分歧标记"之上，做显式的**交叉验证**与**冲突裁决**。
> 单 agent 串行模拟（Stage 7 默认）不具备真独立上下文，不启用本文件——交叉验证的价值正来自"多个互不可见的独立上下文命中同一问题"。

---

## 为什么需要这一层

Phase 2 的"共识（≥2 人提及）"有个盲区：它只数人头，不区分**命中方式**。

- 串行扮演的 N 角色在同一上下文里，前一个角色的印象会污染后一个 → "共识"可能是污染传播，不是独立印证。
- `paper-review-team` 的 N 个角色在**互不可见的独立上下文**里评审 → 若它们命中同一问题，是真正的独立印证，置信度截然不同。

**核心命题**：最致命的硬伤，往往是 ≥2 个独立上下文的角色各自推出同一个结论。单角色质疑可以搁置，多角色独立命中必须当硬伤处理。

> 范例（PaperSwarm med run）：主终点置信区间与报告的 p 值不自洽——魔鬼代言人与统计审稿人在互不读取对方产出的独立上下文里，都按原文数据重算，推出 p 应远小于报告值。这不是某个 agent 随口质疑，而是两个独立视角的交叉验证。

---

## 交叉验证（独立命中 → 置信度升级）

lead 汇总时，对每个被提及的问题，按命中角色数与命中方式定置信度：

| 命中情况 | 置信度 | 处理 |
|---------|--------|------|
| ≥2 个独立上下文角色命中同一问题，且各自给出证据 | 🔴 高置信硬伤 | 必入 P1 路线图 |
| 2 个角色命中，但其中 1 个证据弱 / 仅泛指 | 🟡 中置信 | 入 P2，标注待核 |
| 仅 1 个角色命中（即便措辞尖锐） | 🟢 待核实 | 不自动升级，单列供作者判断 |

**关键**：判定"独立"的依据是角色上下文隔离，不是角色数量。`paper-review-team` 用 delegate 模式 spawn，各角色上下文隔离即视为独立；若退化为同上下文串行扮演，不视为独立（退回 Phase 2 浅层处理）。

## 冲突检测（评价分歧 → 显式裁决）

Phase 2 的"分歧标记"只列出"观点 A vs 观点 B"。本层进一步对冲突**分类、分级、给裁决**。

### 冲突三型

| 类型 | 表现 | 典型场景 |
|------|------|---------|
| 评价相反 | 对同一处，一角色判合格、另一角色判严重缺陷 | 方法论专家认可设计，魔鬼代言人认定致命漏洞 |
| 建议矛盾 | 对同一问题，修改方向相反 | 一方建议扩样本，一方建议换方法 |
| 优先级冲突 | 都认同是问题，但对"多严重"分歧大 | 主编判小修，魔鬼代言人判拒稿 |

### 分级

| 级别 | 影响 | 例子 |
|------|------|------|
| 🔴 严重 | 影响核心结论成立 | 结论的统计基础被质疑 / 核心论据造假嫌疑 |
| 🟡 中度 | 局部论证受损 | 某节文献覆盖不足 / 某个控制变量遗漏 |
| 🟢 轻微 | 风格 / 表述 | 术语不一致 / 图表标注 |

### 裁决规则

每个冲突给：**裁决建议 + 置信度 + 是否升级作者**。

- **可裁决**（有客观依据）：lead 依据证据强度直接给采纳方向，写入 `review-report.md` 的冲突表"采纳"列。
- **不可裁决**（价值 / 风格分歧或证据对等）：标"需作者裁决"，列入 `review-report.md`，由作者在 Stage 8 前拍板。framework 不单独建 `author_resolution.md`（对齐 PaperSwarm 思路但并入汇总报告，避免文件碎片化）。

## 与其他文件的关系

- **`peer-review-simulation.md` Phase 2**：前置。Phase 2 标记共识 / 分歧（浅层），本文件做交叉验证 + 裁决（深层）。Phase 2 是本文件的输入。
- **`review-template.md`**：本文件的方法论产出落到汇总报告的「交叉验证硬伤」段与「冲突」表，格式见该模板（同目录）。
- **Stage 8（`paper-revise-loop`）**：交叉验证出的 🔴 高置信硬伤 = Worker / Verifier 验收时最该盯的项；P1 / P2 / P3 路线图格式（`review-template.md` §5）不变，对 Stage 8 零破坏。
