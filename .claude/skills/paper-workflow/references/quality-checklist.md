# 论文质量检查清单

> 从 5 个维度检查论文质量，配套自评表和常见问题速查。适用于各阶段的质量把关。

---

## 一、5 维度检查

### 维度 1：内容完整性

| 检查项 | 通过标准 |
|--------|---------|
| 摘要 | 五要素齐全（背景/目的/方法/结果/结论） |
| 引言 | 利益陈述、问题缺口、核心概念、方法概述、贡献、结果预告均有体现 |
| 文献综述 | 分类清晰、局限明确、定位句到位 |
| 研究方法 | 研究范式、对象、工具、程序、分析方法的描述完整 |
| 研究结果 | 按研究问题逐一呈现，有数据支撑 |
| 讨论 | 回答问题、理论贡献、实践意义、局限、未来方向 |
| 参考文献 | 引用与正文对应，格式统一 |

### 维度 2：论点清晰度

| 检查项 | 通过标准 |
|--------|---------|
| 核心主张 | 能用一句话说清论文的核心论点 |
| 子主张 | 2–4 个子主张逻辑递进 |
| 证据链 | 每个主张都有对应证据 |
| 论证逻辑 | 没有逻辑跳跃或循环论证 |
| 反面论证 | 考虑了可能的反面解释或替代理论 |

### 维度 3：案例/数据充实度

| 检查项 | 通过标准 |
|--------|---------|
| 量化数据 | 报告了描述性统计、推断统计、效应量 |
| 质性数据 | 有编码示例或典型引用 |
| 案例数量 | 足以支撑论点，不依赖单一案例 |
| 数据呈现 | 表格/图清晰、编号连续、正文有引用 |
| 数据解读 | 每个数据都有文字解读，不是"裸数据" |

### 维度 4：数据真实性

| 检查项 | 通过标准 |
|--------|---------|
| 数据来源 | 能追溯到原始出处 |
| 统计方法 | 方法选择合理，前提条件满足 |
| 引用准确性 | 引文与原文一致，未断章取义 |
| 无编造数据 | 没有为"好看"而调整数据 |
| 证据层级 | 论文定稿中所有内容为 T1 或 T2 |

### 维度 5：语言规范性

| 检查项 | 通过标准 |
|--------|---------|
| 学术用语 | 使用规范学术表达，无口语化表述 |
| 统计符号 | *M*, *SD*, *t*, *p*, *F* 等斜体正确 |
| 格式统一 | 标题层级、编号、引用格式全文统一 |
| 错别字 | 无错字、漏字、多字 |
| 标点规范 | 中英文标点使用正确，半角/全角一致 |
| 字数合规 | 各部分字数在目标期刊要求范围内 |

## 二、质量自评表

每个维度 0–10 分，权重如下：

| 维度 | 权重 | 得分 | 加权得分 |
|------|------|------|---------|
| 内容完整性 | 25% | /10 | |
| 论点清晰度 | 25% | /10 | |
| 案例/数据充实度 | 20% | /10 | |
| 数据真实性 | 15% | /10 | |
| 语言规范性 | 15% | /10 | |
| **总分** | 100% | | **/10** |

**评判标准**：
- 8.5–10：可投稿
- 7.0–8.4：需小幅修改
- 5.5–6.9：需较大修改
- < 5.5：需重写

## 三、常见问题速查表

| 问题类型 | 症状 | 修复方法 |
|----------|------|---------|
| 引言缺缺口 | 读者觉得"这研究不需要做" | Move 2 补充具体的研究空白 |
| 综述成流水账 | "张三研究了……李四研究了……" | 按主题重组，加主题句 |
| 方法不可复现 | 读者看了不知道怎么做的 | 补充研究对象、工具、程序细节 |
| 结果无效应量 | 只报 *p* 值 | 补充 Cohen's *d* 或 η² |
| 讨论与结果重复 | 讨论部分重述数据 | 将数据解读为意义，不重复数字 |
| 结论太空 | "丰富了相关研究" | 改为具体的理论或实践贡献 |
| 引用格式混乱 | GB/T 7714 和 APA 混用 | 统一为目标期刊要求格式 |
| 图表无解读 | 放了表格但正文未讨论 | 每个图表配 2–3 句解读 |
| AIGC 痕迹重 | 大量并列排比、过度对称 | 打破对称结构，增加不对称表达 |

## 四、各阶段完成检查清单

### 选题定位阶段
- [ ] 研究问题明确、可回答
- [ ] 问题有理论或实践价值
- [ ] 目标期刊已确定（至少 2 个候选）
- [ ] 参考文献不少于 15 篇核心文献

### 框架搭建阶段
- [ ] 论文大纲完成（到二级标题）
- [ ] 每章核心论点确定
- [ ] 图表清单规划完成
- [ ] 字数分配合理

### 内容填充阶段
- [ ] 所有章节初稿完成
- [ ] 数据全部到位且已分析
- [ ] 引用文献覆盖充分
- [ ] 图表制作完成

### 学术规范阶段
- [ ] 引用格式统一
- [ ] 统计符号规范
- [ ] 错别字检查完成
- [ ] 证据层级标注已清理（无 T3/T4 残留）

### 质量验收阶段
- [ ] 5 维度检查全部通过
- [ ] 自评表得分 ≥ 7.0
- [ ] 常见问题速查无遗漏
- [ ] 与目标期刊格式要求对照完成

### 定稿输出阶段
- [ ] 格式转换完成（Word/PDF）
- [ ] 双语摘要检查完成
- [ ] 关键词确认
- [ ] 作者信息、基金项目等元数据完整
