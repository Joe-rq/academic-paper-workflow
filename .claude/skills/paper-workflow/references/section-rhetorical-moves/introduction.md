# 引言写作指南：6-Move 结构

> 引言是论文的"橱窗"。好的引言让读者在 2 分钟内理解"为什么要读这篇论文"。
> 本指南将引言拆解为 6 个语步（Move），每个语步有明确的功能和写作要点。

---

## Move 1：利益陈述 — 为什么这个领域重要

**功能**：建立读者关心这个话题的理由。

**写作要点**：
- 从宏观切入，但不要太空泛（"教育很重要"这种话不用说）
- 用数据或事实建立紧迫感
- 1–2 段即可

**示例**：
> 数学是高中阶段的核心学科，承载着培养学生逻辑推理与问题解决能力的重要任务。随着新课程改革推进，高中数学教学正从"教师讲授"向"学生主动探究"转变，但传统课堂普遍存在学生参与度低、高阶思维培养不足的挑战。

---

## Move 2：问题缺口 — 现有研究的结构性局限

**功能**：指出已有研究"差在哪里"，为你的出场铺路。

**写作要点**：
- 不要逐篇评述文献（那是文献综述的工作）
- 聚焦于"研究传统"或"研究范式"层面的局限
- 常用句式："然而，现有研究主要关注……，较少探讨……"
- 用"转折"标志词引导读者：但是、然而、值得注意的是

**示例**：
> 现有研究多从宏观层面分析高中数学教学改革方向，或从单一教学环节探讨课堂改进策略。然而，关于翻转课堂如何促进学生问题解决能力发展的研究尚不充分，尤其缺乏将翻转课堂与问题导向学习相结合的系统探索。

---

## Move 3：核心概念 — 论文的关键理论工具

**功能**：引出你用来解决上述问题的核心概念或框架。

**写作要点**：
- 用 1–2 句话定义核心概念
- 说明这个概念为什么能解决 Move 2 中提出的缺口
- 如果是多个概念，说明它们之间的关系

**示例**：
> 翻转课堂（Flipped Classroom）是将"知识传授"与"知识内化"颠倒的教学模式——学生在课前通过视频自主学习基础知识，课堂时间用于协作探究与问题解决。问题导向学习则提供了"以问题驱动思维"的实践路径，二者结合有望突破高中数学课堂重讲授、轻探究的困境。

---

## Move 4：方法直觉 — 你怎么做的（一段话版）

**功能**：让读者快速了解你的研究路径。

**写作要点**：
- 只需一段话，不要展开细节
- 说明研究类型、对象、核心方法
- 让读者感到"这个研究设计是合理的"

**示例**：
> 本研究采用准实验设计，以某高中两个平行班为研究对象（实验班 45 人、对照班 43 人），在一学期的数学教学中，实验班采用翻转课堂模式，对照班维持传统讲授模式。通过前后测、课堂观察记录和学生访谈收集数据，运用混合方法分析教学效果。

---

## Move 5：贡献陈述 — 这篇论文带来了什么

**功能**：明确列出论文的具体贡献，通常用编号列出。

**写作要点**：
- 3–5 条，每条一个完整贡献
- 用"本文的主要贡献包括"引出
- 每条贡献用"动词 + 内容"格式

**示例**：
> 本文的主要贡献包括：（1）构建了翻转课堂与问题导向学习整合的教学设计框架；（2）通过准实验验证了该框架对高中学生数学学习效果的影响；（3）揭示了翻转课堂促进问题解决能力发展的内在机制；（4）为高中数学教学改革提供了可操作的实践方案。

---

## Move 6：结果预览 — 核心发现的一句话版

**功能**：给读者一个"预告片"，激发阅读兴趣。

**写作要点**：
- 用 1–2 句话概括最核心的发现
- 不要展开数据细节
- 可以适当"卖关子"

**示例**：
> 研究结果表明，实验班学生在问题解决能力和数学思维品质方面显著优于对照班，其中学习基础较弱学生的提升幅度最为显著，为差异化教学策略的设计提供了实证依据。

---

## 引言写作自检清单

| 检查项 | 是/否 |
|--------|-------|
| 读者能在第一段知道这个研究为什么重要吗？ | |
| 明确指出了现有研究的缺口吗？ | |
| 核心概念有清晰定义吗？ | |
| 研究方法有一段话的概述吗？ | |
| 贡献有编号明确列出吗？ | |
| 有核心发现的预告吗？ | |
| 全部引言在 800–1200 字以内吗？ | |
| 没有与文献综述重复的内容吗？ | |

## 常见问题

1. **引言写成了小文献综述**——控制在 Move 2 用 2–3 篇代表性文献点出缺口即可
2. **没有明确的缺口**——如果读者觉得"这个研究已经有人做了"，引言就失败了
3. **贡献太虚**——"丰富了相关研究"不算贡献，要说具体丰富了什么
4. **顺序混乱**——严格按 Move 1→6 的顺序写，不要跳来跳去
