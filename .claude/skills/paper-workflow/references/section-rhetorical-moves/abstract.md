# 摘要写作指南

> 摘要是论文的"微型版"。多数读者只看摘要决定是否阅读全文，检索系统也依赖摘要建立索引。
> 本指南针对中文学术论文摘要的写作规范与技巧。

---

## 摘要结构：五要素

中文学术论文摘要通常包含五个要素，按以下顺序组织：

### 1. 背景（1–2 句）
- 点明研究领域的现实重要性
- 不要写成引言的压缩版，只需 1–2 句建立语境

> 示例：高中数学是培养学生逻辑推理与问题解决能力的核心学科，但传统课堂普遍存在学生参与度低、高阶思维发展不足的困境。

### 2. 目的（1 句）
- 明确说明本研究要解决什么问题
- 用"本文旨在……""本研究以……为目的"引出

> 示例：本文旨在探究翻转课堂在高中数学教学中的应用效果及作用机制。

### 3. 方法（2–3 句）
- 研究设计类型
- 研究对象与样本量
- 核心方法或工具

> 示例：采用准实验设计，以某高中 88 名一年级学生为研究对象，实验组接受一学期翻转课堂教学，对照组维持传统讲授模式，通过前后测和访谈收集数据。

### 4. 结果（2–3 句）
- 最核心的 2–3 个发现
- 包含关键数据（显著性、效应量）
- 不需要展开每个维度的细节

> 示例：实验组学生的问题解决能力显著高于对照组（p < 0.01, d = 0.85），其中问题表征维度提升最为显著。访谈数据显示课前自主学习的充分性与课堂探究深度正相关。

### 5. 结论（1–2 句）
- 研究的核心结论和实践价值
- 不要引入新信息

> 示例：翻转课堂教学模式能有效提升高中数学教学质量，对学习基础较弱学生的促进效果尤为显著，为高中数学教学改革提供了实证依据和可操作的实践方案。

## 字数规范

| 类型 | 字数要求 |
|------|---------|
| 中文摘要 | 200–300 字 |
| 英文摘要 | 150–250 words |
| 会议短文摘要 | 100–200 字 |

---

## 关键词规范

- 数量：3–5 个
- 来源：从标题和摘要中提取
- 顺序：按重要性或逻辑顺序排列
- 格式：用分号分隔

> 示例：翻转课堂；高中数学；问题解决能力；教学效果；准实验研究

**选词原则**：
- 优先使用学科规范术语
- 避免过于宽泛的词（如"教育""研究"）
- 避免过于狭窄的词（如某个学校的名称）
- 检查关键词是否覆盖研究的主要维度

---

## 双语摘要注意事项

中文和英文摘要不是简单的翻译关系：

1. **术语一致性**：中文术语在英文中有固定译法的，采用公认译法
2. **时态差异**：中文摘要不严格区分时态，英文摘要通常用过去时报告研究过程和结果，用现在时陈述结论
3. **语态差异**：中文倾向主动语态（"本研究发现……"），英文可适当使用被动语态
4. **句式差异**：中文多用短句串联，英文可适当使用从句
5. **数量一致**：双语关键词的顺序和数量应一致

## 摘要写作自检清单

| 检查项 | 是/否 |
|--------|-------|
| 五个要素（背景/目的/方法/结果/结论）齐全吗？ | |
| 字数在 200–300 字以内吗？ | |
| 包含核心数据（显著性、效应量等）吗？ | |
| 关键词 3–5 个，用分号分隔吗？ | |
| 没有引用参考文献吗？ | |
| 没有使用缩写（首次出现时）吗？ | |
| 读者看完摘要能准确复述核心发现吗？ | |
| 英文摘要不是中文的逐字翻译吗？ | |

## 常见问题

1. **背景写太多**——背景 1–2 句足矣，不要写成小引言
2. **方法写太细**——摘要中的方法只需关键信息，不报具体工具版本号
3. **结果没有数据**——"效果显著"不如"*p* < 0.001, d = 0.85"
4. **结论太笼统**——"有一定参考价值"等于什么都没说
5. **关键词与标题重复**——关键词应补充标题未覆盖的重要维度
