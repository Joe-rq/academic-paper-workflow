# 研究方法写作指南：5-Move 结构

> 研究方法的核心标准是"可复现性"——读者看了你的方法部分，应该能判断研究设计的合理性，甚至能在类似情境中复现。
> 本指南将方法部分拆解为 5 个语步。

---

## Move 1：方法概述 — 研究的整体设计

**功能**：给读者一个"全景图"，看清研究设计的骨架。

**写作要点**：
- 开篇说明研究范式（定量/定性/混合方法）及理由
- 交代研究设计的类型（实验/准实验/调查/案例研究/行动研究等）
- 用一张研究框架图辅助说明（推荐）

**示例**：
> 本研究采用解释性顺序混合方法设计（explanatory sequential mixed methods），第一阶段通过准实验收集量化数据以评估教学效果，第二阶段通过半结构化访谈收集质性数据以解释量化结果背后的机制。

---

## Move 2：方法论证 — 为什么选择这种方法

**功能**：说服读者"这个方法是最适合回答研究问题的"。

**写作要点**：
- 回应研究问题，说明方法与问题的匹配度
- 如果有多种可选方法，简要说明为什么没选其他方法
- 引用方法论文献支撑你的选择

**示例**：
> 选择混合方法设计基于以下考虑：本研究不仅需要量化证据验证教学效果（适合准实验设计），还需要理解效果背后的学习机制（适合访谈法）。单纯量化方法无法揭示"为什么有效"的过程机制，单纯质性方法则难以提供普适性的效果证据。Creswell & Plano Clark（2018）指出，解释性顺序设计特别适合"先验证效果、再解释机制"的研究目的。

---

## Move 3：实施步骤 — 具体的研究过程

**功能**：详细描述"你是怎么做研究的"，保证可复现性。

**各要素的写作要点**：

### 研究对象
- 抽样方法、样本量、人口学特征
- 纳入/排除标准
- 回应率/流失率

### 研究工具
- 成熟量表：注明来源、信效度指标
- 自编工具：说明编制过程、预试结果
- 访谈提纲：附在附录或说明核心问题

### 研究程序
- 按时间顺序描述
- 关键节点（干预开始、数据收集时点等）要明确
- 用流程图辅助说明（如有必要）

**示例**：
> 研究对象为某市普通高中一年级两个平行班，共 88 名学生。采用便利抽样，以自然班为单位分组：一班为实验组（n=45），二班为对照组（n=43）。两组在入学成绩、性别比例上无显著差异（p > 0.05）。研究周期为一学期，实验组接受翻转课堂教学，对照组维持传统讲授模式。前测在第 1 周实施，后测在学期末实施。

---

## Move 4：关键决策 — 核心的研究参数或选择

**功能**：突出研究中的关键设计决策及其依据。

**写作要点**：
- 变量的操作化定义（如"问题解决能力"如何测量）
- 分析方法的选择及理由
- 显著性水平、效应量标准等参数
- 质性分析的编码策略（如主题分析、扎根编码）

**示例**：
> "问题解决能力"采用自编测试卷测量，包含问题表征（30%）、策略选择（40%）和反思监控（30%）三个维度。该试卷经 3 位学科专家审阅，预试（n=30）的 Cronbach's α 为 0.82。量化数据采用 SPSS 26.0 进行分析，主要使用独立样本 t 检验和协方差分析（ANCOVA），以 Cohen's d 报告效应量。

---

## Move 5：局限应对 — 方法局限及应对措施

**功能**：主动承认方法局限，并说明你的应对策略。体现学术诚实。

**写作要点**：
- 选择最关键的 2–3 个局限，不要面面俱到
- 每个局限配一个应对措施
- 不要把局限写得好像"研究不靠谱"——要写得像"我们有自知之明"

**示例**：
> 本研究存在以下局限：第一，采用便利抽样而非随机抽样，样本代表性受限。对此，我们通过选择在入学成绩和人口学特征上具有同质性的两个平行班来控制选择偏差。第二，仅在一所学校开展研究，外部效度有限。后续研究可在多所学校复制本研究设计以增强结论的推广性。

## 写作自检清单

| 检查项 | 是/否 |
|--------|-------|
| 研究范式的选择有明确理由吗？ | |
| 研究对象描述完整吗？（样本量、特征、抽样方法） | |
| 研究工具的信效度有报告吗？ | |
| 研究程序按时间顺序清晰描述了吗？ | |
| 变量的操作化定义清楚吗？ | |
| 分析方法的选择有理由吗？ | |
| 局限有具体说明且配了应对措施吗？ | |
| 方法部分的内容量与结果部分匹配吗？ | |
