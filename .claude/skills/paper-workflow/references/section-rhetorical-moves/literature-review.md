# 文献综述写作指南：3-Move 结构

> 文献综述不是"文献罗列"，而是"有立场的学术对话"。本指南将综述拆解为 3 个语步，帮助研究者系统梳理已有研究并建立自己的学术定位。

---

## Move 1：类别聚类 — 将已有研究分为 2–4 类

**功能**：建立文献的"地图"，让读者看到这个领域的全景。

**分类方法**：
- 按研究视角分（如：宏观政策视角、微观教学视角、技术赋能视角）
- 按理论框架分（如：建构主义取向、认知主义取向、社会文化取向）
- 按研究方法分（如：量化研究、质性研究、混合方法研究）
- 按研究对象分（如：基础教育段、高中教育段、职业教育段）

**写作要点**：
- 每个类别用 2–4 篇代表性文献支撑
- 用主题句开头："关于 X 的研究，大致可分为以下几类……"
- 不要按时间顺序罗列，要按逻辑关系组织

**示例**：
> 关于翻转课堂的研究，可按研究焦点分为三类：一是教学模式的构建研究，侧重于……；二是教学效果的实证研究，聚焦于……；三是学科教学的迁移应用，探索翻转课堂在具体学科中的实践形态。

---

## Move 2：逐类局限 — 每类的结构性局限

**功能**：对每个类别进行"批判性审视"，指出其尚未解决的问题。

**写作要点**：
- 不是贬低已有研究，而是指出"还可以走多远"
- 用"虽然……但……"的句式体现学术礼貌
- 局限要有具体证据，不要笼统说"研究不够深入"
- 最后要归纳出贯穿各类别的共同局限

**示例**：
> 第一类研究虽在教学流程设计上提供了可参考的框架，但多停留在经验总结层面，缺乏对学习机制的理论阐释。第二类实证研究提供了效果证据，但多采用短期单次实验，难以观察持续性影响。第三类应用研究丰富了实践样态，但较少关注学科特有的高阶思维发展问题。综合来看，现有研究普遍缺乏将翻转课堂与学科问题解决能力进行系统整合的理论尝试。

---

## Move 3：定位句 — 一句话定位本文

**功能**：在综述的结尾，用一句话宣布"本文要做什么"。

**写作要点**：
- 通常是一段话，不是一句话字面意思
- 格式：鉴于上述局限 + 本文的切入点
- 与引言中的缺口陈述呼应，但更加具体

**示例**：
> 鉴于现有研究在翻转课堂与学科问题解决能力整合方面的不足，本文尝试构建"翻转课堂的问题导向教学"框架，并通过准实验研究检验其对高中数学教学效果的影响，以期弥补上述研究缺口。

---

## 综述结构模板

```markdown
## 文献综述

### 1. [类别 A 名称]（~400 字）
- 主题句：界定这一类研究
- 代表文献：2–3 篇核心发现
- 小结：这类研究的贡献

### 2. [类别 B 名称]（~400 字）
- 同上结构

### 3. [类别 C 名称]（~400 字）
- 同上结构

### 4. 研究述评与本文定位（~300 字）
- 各类局限的归纳
- 定位句：本文的切入点
```

## 写作自检清单

| 检查项 | 是/否 |
|--------|-------|
| 文献分为 2–4 个有逻辑的类别了吗？ | |
| 每个类别有代表性文献支撑吗？ | |
| 指出了每个类别的具体局限吗？ | |
| 归纳了贯穿各类别的共同局限吗？ | |
| 有明确的定位句引出本文吗？ | |
| 没有简单地按时间顺序罗列文献吗？ | |
| 综述与引言不重复吗？（引言点缺口，综述展开缺口） | |
| 引用格式统一规范吗？ | |

## 常见问题

1. **流水账式综述**——"张三研究了……李四研究了……"是最差的写法，要用主题句串联
2. **只有正面评价**——文献综述必须有批判性视角，否则无法确立研究空间
3. **与引言内容重复**——引言用 2–3 句话点出缺口，综述展开论证这个缺口
4. **定位句缺失**——综述的最后一段必须回到"所以本文要做什么"
5. **文献太少或太多**——一般 15–30 篇，核心文献 8–12 篇
