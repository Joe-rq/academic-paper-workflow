# 讨论写作指南：5-Move 结构

> 讨论是论文的"升华区"——从数据出发，上升到理论和实践层面。好的讨论让读者看到你的研究"为什么重要"以及"下一步可以做什么"。

---

## Move 1：回答研究问题 — 研究发现如何回应核心问题

**功能**：开门见山，用研究结果直接回答引言中提出的研究问题。

**写作要点**：
- 回到研究问题，逐一用结果回答
- 不要重复罗列数据（结果部分已做），而是"解读"结果
- 用"本研究发现……这一发现表明……"的句式

**示例**：
> 本研究旨在探究翻转课堂对高中数学学习效果的影响。结果表明，实验组学生在问题解决能力和数学思维品质上均显著优于对照组，这表明将翻转课堂与问题导向学习相结合是提升高中数学教学质量的有效路径。

---

## Move 2：理论贡献 — 对理论的推进

**功能**：说明你的研究发现如何推进了现有理论。

**写作要点**：
- 与文献综述中的理论框架呼应
- 说明你的发现支持了、修正了还是挑战了已有理论
- 贡献要具体，不要说"丰富了理论体系"这种空话

**常见贡献类型**：
- 验证了某理论在新情境（新学段、新学科）中的适用性
- 发现了已有理论未预见的新机制
- 将两个理论框架整合，产生了新的解释力
- 揭示了边界条件（在什么条件下理论成立/不成立）

**示例**：
> 本研究对 Garrison、Anderson & Archer 的"探究社区"（Community of Inquiry）理论在高中数学情境中的适用性提供了实证检验。研究发现，该理论框架在数学学科中需要强化"认知临场感"与"教学临场感"的衔接——当课前自主学习与课堂探究任务紧密对应时，问题解决能力的促进效果显著增强。这一发现拓展了原理论在数学教学领域的解释力。

---

## Move 3：实践意义 — 对教育实践的启示

**功能**：说明研究对一线教师、管理者、政策制定者的实际价值。

**写作要点**：
- 分层次说明：课堂层面、学校层面、政策层面
- 建议要具体可操作，不要说"应重视 XX"这种废话
- 标明建议的证据基础（基于哪个发现）

**示例**：
> 本研究对教学实践的启示包括：（1）高中数学教师可在单元设计中引入"课前自学→课堂探究→反思提炼"的教学流程，本研究发现该流程能显著提升学生的问题解决能力；（2）教师应特别关注课前学习任务与课堂探究的衔接设计，数据分析显示这一要素对学习基础较弱学生的促进效果最为显著；（3）学校教学管理部门可在教研活动中组织教师开展翻转课堂教学设计的专题研讨。

---

## Move 4：局限分析 — 诚实且具体

**功能**：展现学术诚信，帮助读者准确评估结论的适用边界。

**写作要点**：
- 选择 3–4 个最关键的局限
- 每个局限：说明是什么 + 对结论的可能影响 + 后续如何改进
- 不要写"本研究不够完善"这种无信息量的局限
- 不要写太多局限以至于读者怀疑研究的价值

**示例**：
> 本研究存在以下局限。第一，研究仅在一所城市普通高中进行，样本同质性较高，结论向其他类型学校的推广需谨慎。第二，研究周期为一学期，属于短期效果考察，翻转课堂的长期效果尚需追踪研究。第三，本研究的质性数据主要依赖学生自我报告，未来研究可引入课堂观察或学习分析等更客观的数据源。

---

## Move 5：未来方向 — 后续研究建议

**功能**：为后续研究者指明方向，体现你对领域的深入思考。

**写作要点**：
- 2–3 个方向即可
- 每个方向说明"为什么值得做"和"大致怎么做"
- 可以从 Move 4 的局限中自然引出

**示例**：
> 后续研究可沿以下方向推进：（1）在多所学校开展复制研究，采用随机对照设计，增强结论的外部效度；（2）延长研究周期至一个学年或更长，考察翻转课堂的持续效应和衰减效应；（3）引入过程性数据（如课堂录像分析、学习轨迹追踪），深入揭示翻转课堂促进问题解决能力发展的微观机制。

---

## 讨论与各部分的关系

```
引言 → 提出研究问题 ─────────────→ 讨论 Move 1 回答问题
文献综述 → 建立理论框架 ──────────→ 讨论 Move 2 理论推进
方法/结果 → 实证证据 ─────────────→ 讨论 Move 3 实践转化
方法 → 设计局限 ─────────────────→ 讨论 Move 4 局限分析
所有部分 → 未解决的问题 ──────────→ 讨论 Move 5 未来方向
```

## 写作自检清单

| 检查项 | 是/否 |
|--------|-------|
| 开篇直接回答了研究问题吗？ | |
| 理论贡献具体且有证据支撑吗？ | |
| 实践建议可操作且有数据依据吗？ | |
| 局限分析诚实具体，且有改进方向吗？ | |
| 未来方向从局限中自然引出吗？ | |
| 没有引入新的数据或新文献吗？ | |
| 篇幅与结果部分大致匹配吗？ | |
